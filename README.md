## Overview

This repository contains coursework and projects completed for ASU's ACO/MAT494 — Foundations of Artificial Intelligence, part of Ryan's Bachelor's in Applied Computing (Cybersecurity). The repo is organized as a collection of Git submodules, each pointing to a dedicated project repository.

---

## Projects

### 1. AI Pokémon Classification (Final Project)
**Submodule →** [Pokemon-Type-Classification-from-Image-ACO494_Final](https://github.com/wildinry/Pokemon-Type-Classification-from-Image-ACO494_Final)

A machine learning / computer vision project that classifies Pokémon types from images. This was the final project for the course, applying image classification techniques learned throughout the semester.

**Key concepts:** Image classification, supervised learning, convolutional neural networks (CNN)

---

### 2. Wumpus World AI Agent
**Submodule →** [WumpusWorld_ACO494](https://github.com/wildinry/WumpusWorld_ACO494)

An implementation of the classic Wumpus World problem — a knowledge-based AI agent that navigates a grid environment using logical inference to find gold and avoid hazards (pits and the Wumpus).

**Key concepts:** Knowledge representation, propositional logic, inference, intelligent agents

---

### 3. Custom MNIST Digit Recognition
**Submodule →** [CustomMNIST-Digit-Recognition-ACO494](https://github.com/wildinry/CustomMNIST-Digit-Recognition-ACO494)

A neural network implementation trained on the MNIST handwritten digit dataset, with customizations applied for the course project.

**Key concepts:** Neural networks, deep learning, digit recognition, training/evaluation pipelines
